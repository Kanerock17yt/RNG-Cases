# RNG Cases
This is a game in the terminal made in python.

## Things you need to install
 - [Python](https://www.python.org/)

## How to install
### 1. Command Prompt
1. Open command prompt without admin perm and type this command (change YOURUSERHERE with your current user!):
```
cd "C:\Users\YOURUSERHERE\Downloads"
```
2. Open cmd (without admin permission) and type in this command:
```
git clone https://github.com/Kanerock17yt/RNG-Cases
```
### 2. Download it from the github page
1. go to the [releases](https://github.com/Kanerock17yt/RNG-Cases/releases)
2. click on the newest release
3. click on the .zip and it should be downloaded/downloading.
